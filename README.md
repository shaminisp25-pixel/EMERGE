# EMERGE Companion

A Gen-Z friendly emotional companion designed to help you navigate your feelings. This project provides a personalized experience through mood tracking, journaling, and interaction with a virtual pet.

## Quick Start (Local Development)

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- npm

### Setup

1. Clone the repository:
```bash
git clone <repository-url>
cd my-emerge-companion
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:8080`.

## Features

- **Personalized Onboarding**: Tailored questions to understand your vibes and preferences.
- **Virtual Pet**: Select and interact with a companion that grows with you.
- **Mood Tracking**: Log your feelings and view emotional trends.
- **Journaling**: Reflect on your day with text and sticker-based entries.
- **Personalized Insights**: Visualizations and quotes based on your mood.

## Technologies

- **Vite**: Next-generation frontend tooling.
- **React**: A JavaScript library for building user interfaces.
- **TypeScript**: Typed JavaScript for better development experience.
- **Shadcn UI**: Beautifully designed components.
- **Tailwind CSS**: A utility-first CSS framework.
